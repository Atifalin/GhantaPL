# GhantaPL
A web app built using Python and Django to act like an auctioneer produces Players from FIFA 21/22 to 4 human players to bid on and edit teams on FIFA 21/22 to play with these teams

Here's what i have imagined for this GhantaPL - Ghanta Premier League

Me and my 3 brothers have devised a way to play FIFA games with each other on a PS4, A game mode that I created, Its Called The Ghanta Premeier League
it was created in 2 phases 
the first phase was were we had 4 categories of players : GK, DEF, MID, FWD and in each we had 6-16 or more players for those positions.
we would randomly pick one each from a pot and create custom teams on FIFA and play League matches. 
This was really fun for us so we did this for months

Phase 2 began when this got a bit boring for 2 of our brothers as there was too much randomness in this.
Then i deciced that we should bid for these players and the highest spender gets the player. 
for this i set up a few rules.
List of Players shouldnt be position wise but tier wise
Teir 1 (Top Tier) had players like Messi Ronaldo and other top players
Teir 2 had players that are very good but not above 85 overall
Teir 3 had Players ranging from 75-84 overall skill
Teir 4 was substitutes for this we decided who to add to this tier regardless of their over skill (for example in fifa 19 Rashford was 79 but was added to t4 for his pace)
